Un sistema autónomo que conecta clientes, administración y técnicos de forma automática, asegurando trazabilidad total, evidencias en tiempo real y reportes instantáneos sin intervención manual.

📌 Dolor de negocio

En servicios de campo, la información se dispersa: registros manuales incompletos, técnicos que no reportan evidencias, clientes que desconocen el estatus y administración que pierde tiempo coordinando por llamadas y correos.

Esto genera pérdidas económicas, baja confianza y poca escalabilidad operativa.

💡 Cómo funciona Assignia

1️⃣ El cliente llena un formulario web para solicitar información o agendar un servicio.
2️⃣ El sistema envía una alerta automática a ventas y administración.
3️⃣ El equipo cierra la venta y desde el panel asigna al técnico ideal.
4️⃣ El técnico recibe una alerta con enlace personalizado con datos del cliente, ruta y evidencias requeridas.
5️⃣ El cliente recibe un enlace para monitorear su servicio en tiempo real.
6️⃣ Al finalizar, el técnico llena el formulario de cierre, sube fotos y automaticamente actualiza el estatus.
7️⃣ El cliente y administración reciben un PDF detallado del servicio realizado.

🚀 Impacto real

✔️ Cero pérdida de información
✔️ Supervisión en tiempo real
✔️ Evidencias fotográficas centralizadas
✔️ Clientes informados y satisfechos
✔️ Reducción de llamadas, correos y tareas repetitivas
✔️ Reportes PDF listos para auditorías

📂 Ficha técnica

- Herramientas: Make, Notion/Airtable, Formularios HTML, JavaScript, API PDF
- Roles: Cliente, Administrativo, Técnico
- Integración: Notificaciones automáticas por correo y Telegram
- Tiempo de entrega: 14 días
- Casos de uso: Instalaciones, mantenimientos, reparaciones, auditorías en campo.
